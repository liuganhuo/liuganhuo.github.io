Windows下使用GUDA配置GPU并行环境

  • 操作系统版本：Win10
  • Visual Studio版本： 2022
  • CUDA版本：CUDA Toolkit 12.1

CUDA安装

查看显卡驱动，确定合适版本CUDA

从官网下载CUDA，完成后查看当前CUDA是否成功安装

下载Visual Studio（对照网上教程）

Visual Studio配置

1.创建项目

选择空项目即可

2.调整配置管理器平台类型

右键项目→ 属性→ 平台“x64”

3.配置生成属性

右键项目 → 生成依赖项→ 生成自定义→ 勾选“CUDA XXX”

4.配置基本库目录

右键项目→属性→ 配置属性→ VC++目录→ 包含目录，添加以下目录：

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.1\include

……→ 库目录，添加以下目录：

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.1\lib\x64

5.选用CUDA静态链接库

右键项目→ 属性→ 配置属性→ 链接器→ 输入→ 附加依赖项，将lib\x64中所有.lib文件加入此项中
